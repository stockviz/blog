# Macro: Using Currencies to Predict NIFTY, Part I

**Blog post:** [Macro: Using Currencies to Predict NIFTY, Part I](https://stockviz.biz/index.php/?p=2092543)

> Trained a Support Vector Machine on DTWEXM currency index returns to predict NIFTY 50 weekly returns. A 3rd-degree polynomial kernel produced the best results on the test set, but the model failed to predict the 2018 NIFTY correction as USDINR moved independently of DTWEXM during that period.
