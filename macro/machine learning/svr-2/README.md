# Macro: Using Currencies to Predict NIFTY, Part II

**Blog post:** [Macro: Using Currencies to Predict NIFTY, Part II](https://stockviz.biz/index.php/?p=2092783)

> Explored tuning the polynomial degree parameter for the SVM model predicting NIFTY from currency indices. Found no single degree consistently outperformed across different training datasets, and that using a 2005–2018 dataset (degree 4) sidestepped the 2018 correction while the 2000–2018 dataset did not, highlighting that older data can sometimes be counter-productive.
