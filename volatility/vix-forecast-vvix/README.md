# VVIX for VIX

**Blog post:** [VVIX for VIX](https://stockviz.biz/index.php/?p=40890477)

> Applied a two-stage volatility-of-volatility forecasting method (from Markku Kurtti) to VIX prediction, separating average volatility and volatility-of-volatility forecasts. LOCF again beat the model at both 1-day and 20-day horizons, failing even at the mean-reversion prediction where models should theoretically have an edge.
