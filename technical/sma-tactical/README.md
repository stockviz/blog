# SMA Strategies using ETFs

**Blog post:** [SMA Strategies using ETFs](https://stockviz.biz/index.php/?p=2099253)

> Backtested simple moving average (SMA) toggle strategies on Indian ETFs (NIFTY 50, MIDCAP 100, SMLCAP 100). Found shorter SMA lookbacks boost performance but increase trade frequency; the strategy beats actively managed midcap funds even after transaction costs, though scalability is constrained by ETF liquidity and premium/discount to NAV issues.
