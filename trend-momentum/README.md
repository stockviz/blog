# Trending Momentum

**Blog post:** [Trending Momentum](https://stockviz.biz/index.php/?p=40889073)

> Investigated whether simple moving-average overlays can time momentum indices (NIFTY MIDCAP150 MOMENTUM 50 TR and NIFTY200 MOMENTUM 30 TR). Found that short-lookback SMAs help reduce drawdowns, and the best config from 2010–2015 showed promise out-of-sample (2016–2022), though after-cost returns would be lower.
