# Mahalanobis Distance

**Blog post:** [Mahalanobis Distance](https://stockviz.biz/index.php/?p=40890153)

> Constructed a portfolio switching between equities and bonds based on the Mahalanobis distance between them, inspired by a financial turbulence index paper. Out-of-sample results with transaction costs were promising but did not clearly beat simpler drawdown-avoidance strategies, though midcap outperformance was notable.

## Summary

computes Mahalanobis distance as a multivariate anomaly detection metric for market regimes

### `mahalanobis-dist.R`
