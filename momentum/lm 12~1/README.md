# Linear Model Momentum

**Blog post:** [Linear Model Momentum](https://stockviz.biz/index.php/?p=40890417)

> Tested whether linear regression-based momentum (predicting next-month returns from recent 12-month and 1/3/6-month performance) beats naïve momentum. Counter-intuitively, the simple naïve momentum strategy outperformed linear models, reinforcing that the simplest approach is often best.
