# Intraday Momentum, an Update

**Blog post:** [Intraday Momentum, an Update](https://stockviz.biz/index.php/?p=40890432)

> Revisited the intraday momentum strategy (first half-hour return predicts last half-hour) eight years after an initial sniff test. Confirmed it continues not to work on NIFTY, BANK NIFTY, and MID SELECT across both 15-min and 30-min formations, illustrating that widely known strategies tend to stop working.
